# Meme Asset Sourcing — Neckbeard

**Dave's workflow:**
- **Audio:** Go to the myinstants.com link → right-click the orange play button → "Save Audio As" → save as `<id>.mp3` → drop in `game/assets/memes/audio/` → add id to `NB.MEME_AUDIO` in `memes.js`
- **Images:** Go to the PNG site link → download the transparent PNG → save as `<id>.png` → drop in `game/assets/memes/img/` → add id to `NB.MEME_ART` in `memes.js`
- Images you can also just screenshot + background erase yourself — whatever's faster.
- Until an asset is dropped in, the game shows a placeholder badge + synth audio. Nothing breaks.

## IP notes (your call, not advice)
The audio clips on myinstants are user-uploaded and the site has been up for 10+ years serving these exact files — for a non-commercial hackathon, the risk is minimal. The 🔴 ones to be most aware of in a *public* submission: Nyan Cat (creator won a lawsuit vs a game studio), Keyboard Cat (creator sued Warner Bros), Pepe (Matt Furie actively enforces). Everything else is effectively community property at this point for hackathon use.
